## 原文内容
华为超节点需求旺盛、三星电子DRAM或涨价20%、DeepSeek-V4正式版计划于7月中旬正式上线并引入峰谷定价机制

🔥计算
#华为超节点Atlas 950 SuperPoD需求旺盛
▪ 6月29日讯，华为Atlas 950 SuperPoD——国内唯一的液冷千卡超节点，单柜最大支持8192张昇腾NPU高速互联，2026Q4正式上市。当前订单或已排满，主要客户为互联网云厂商。（极客网）

＃Meta筹建云服务业务、计划对外售卖闲置AI算力与模型
▪7月1日讯，Meta正规划一项全新的云基础设施业务，计划对外提供AI算力和模型的使用权。单一事件并不能证明AI行业就此走向衰退，更多是大厂开始进行精细化资产管理与回补现金流的防守性策略。（科创板日报）

🔥网络/存储
#Meta通过CXL复用拆机内存
▪6月30日讯，The Register博文称Meta为减少采购新硬件需求发布Vistara定制芯片方案，每个Vistara ASIC集成了两个独立的72-bit DDR4内存通道，支持高达3200MT/s的速度，每个芯片最大容量可达256GB。（IT之家）

#英特尔消费级与服务器CPU涨价
▪7月5日讯，英特尔提高了部分消费级和服务器CPU的价格，系成本上升以及需求飙升，部分处理器价格上涨了30至50美元，数据中心级产品的价格上涨了数百甚至数千美元。（半导体行业纵横）

#三星电子DRAM或涨价20%
▪7月3日讯，据ZDnet Korea报道，三星电子正在与客户进行谈判，目标在2026Q3将通用DRAM的ASP环比提高20%。（科创板日报）

🔥模型/应用
#DeepSeek-V4正式版计划于7月中旬正式上线
▪6月30日讯，DeepSeek V4 正式版计划于 7 月中旬正式上线，正式版将进一步优化先关性能。 同时，V4正式版发布后将同步调整 API 定价策略，引入峰谷定价机制, API 高峰时段价格将是平时价格的 2 倍。（财联社）

#GPT-5.6有望7月7日上线
▪7月4日讯，GPT-5.6有望7月7日上线，分为Sol Ultra、Terra和Luna三大版本，主要以低价和高推理效率对标Claude 5。其中Sol Ultra拟对标Fable 5，凭借高token生成效率价格较Fable便宜两倍，同时向用户开放更大token额度。（新智元）

#Seedance2.5预计7月上线
▪6月30日讯，Seedance 2.5主打影视级的精细度与高可控性，最高可50个将文本、图片、音频及等参考素材转化为高度可控的分镜片段，面向电商广告、社交媒体和视觉叙事等专业内容生产赛道，最长生成时长180s，已于字节海外官网更新介绍。（界面新闻）
