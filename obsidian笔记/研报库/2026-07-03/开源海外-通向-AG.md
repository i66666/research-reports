## 原文内容
📰 【开源海外 | 通向 AGI】2026 年 07 月 02 日 AI 大模型晚间速递
#OpenAI：
据 FT/CNBC 报道，公司正与白宫讨论向美国政府捐赠 5% 股权，按近期 8520 亿美元估值计算约 426 亿美元；Altman 称这是与公众分享 AI 红利的最佳方式，已被纳入其 4 月提出的 "Public Wealth Fund" 设想雏形。（FT / CNBC / Yahoo Finance | 7 月 2 日）
#Meta：
据 Bloomberg 报道，公司正筹划云基础设施业务 "Meta Compute"，将富余 AI 算力与模型访问对外销售，直接对标 AWS / Google Cloud / Azure；新业务由基础设施主管 Santosh Janardhan、超智实验室负责人 Daniel Gross 与总裁 Dina Powell McCormick 牵头，
可能效仿 CoreWeave 出售裸计算、模仿 AWS 托管 AI 模型（含闭源 Muse Spark）；此前已承诺未来几年投入 1829 亿美元建设 AI 基础设施，俄亥俄州超大型数据中心年内上线。（Bloomberg / TechCrunch | 7 月 1 日）
#Google：
2025 年公司年度用电量同比 +37%、创历史最大增幅，数据中心全年耗电超 4200 万 MWh，已超过新西兰 / 丹麦 / 尼日利亚全国用电量；自 2019 年以来累计增长超 250%，AI 基础设施建设速度已超过电网脱碳速度。（Ars Technica | 7 月 2 日）
#月之暗面：
Kimi K2.7 Code 开源权重模型在 GitHub Copilot 全面可用，成为 Copilot 模型选择器首个可选开源权重模型，逐步向 Pro / Pro+ / Max 计划推送，已支持 VS Code 1.127+、Visual Studio 17.14.6+、JetBrains 1.9.1-251+、Xcode、Eclipse 及 CLI，
由 GitHub 托管于 Azure 按供应商列表价按量计费。（GitHub Changelog | 7 月 2 日）
#行业动态：
证监会同意宇树科技科创板 IPO 注册申请，宇树为国内头部民用足式 / 人形机器人企业，2016 年王兴兴在杭州创立、截至 2025 年 6 月员工超 1000 人，核心零部件（关节电机、减速器、控制器）全栈自研、成本约为进口产品 1/3，四足机器人全球销量领先。（IT 之家 | 7 月 2 日）
