## 原文内容

🥇【天风传媒海外】强制通用 Agent 范式切换第二幕：C 端活跃用户如期破千万，B 端 Agent 产品提速🚀

🔺背景：OpenAI 通用 Agent（Codex+ChatGPT Work）活跃用户于 7 月 21 日突破 1000 万、约一周翻倍(我们7/15电话会提示)。7 月 22 日 OpenAI 基于 B 端场景推出企业 Agent 套件 Presence，两点看：

1️⃣ 是什么 → 企业 Agent 运行时：连接企业内部数据、政策、存量软件与工作流，首批落地客服、销售、账单、保险理赔与 IT 工单等强流程场景。
2️⃣ 怎么交付 → 内置仿真测试、guardrails、人工审核与 AI 自动打分，上线后可用 Codex 排障迭代；当前 case-by-case 开放、需 OpenAI 工程师或指定伙伴进场，#形态是产品加FDE交付、针对企业生产系统。

🌋落点：付费口径的迁移是关键信号，#企业Agent的收费将从token走向seat、workflow与outcome，收入质量高于 API、粘性驱动估值逻辑重估。对海外应用层（客服/CRM/ITSM SaaS）是竞合压力；映射国内，#B端企业Agent是国内大厂的下一张明牌，腾讯（企业微信+WorkBuddy）与阿里（钉钉+千问办公）掌握企业工作流入口，模型侧依旧坚定推荐具备卡位的模型公司。

🌋一句话：C端基于强制范式切换用户数量已经翻倍，等待观察用户留存。#B端产品更新提速企业Agent渗透、OpenAI基于模型能力向更粘性的企业软件层迁移。

建议关注：
1、新场景的确立：#OpenAI三季度ARR增长超预期的确定性提升、B端C端同时推动通用Agent下半年增长提速；
2、具备卡位的模型公司：智谱、MiniMax；
3、具备分发能力的 Agent 产品商：腾讯控股、阿里巴巴。
