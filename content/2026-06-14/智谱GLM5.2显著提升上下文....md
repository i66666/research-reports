## 原文内容

智谱GLM5.2显著提升上下文长度，海外模型出口管制强化主权AI趋势【东吴传媒互联网&海外张良卫团队】

事件：

1、智谱周六正式发布最新旗舰模型GLM-5.2。1）官方尚未单独披露具体参数规模、正式技术报告。2）上线即面向GLM Coding Plan 全量用户正式开放，API将在下周上线，下周将全面开源，遵循MIT协议。3）1M上下文是本次升级重点之一，GLM-5.1上下文长度为 200K tokens，模型超过一定上下文对前文细节的检索和理解能力会明显下降，此次上下文长度显著提升将有助于提升模型的长程任务处理能力，为构建复杂的 Agent 智能体应用提供坚实的底层支撑。

2、Anthropic Claude Fable 5和 Mythos 5于6月9日发布。1）社区普遍认为Fable 5是目前公开可用模型中最强的编码/工程模型，尤其适合复杂、多文件、长周期任务。2）价格与消耗极高是主要批评，当前价格在输入 $10/M tokens，输出 $50/M，同时因安全/出口管制暂停访问。3）主权AI趋势得到强化，6月3日，欧盟委员会公布“欧洲技术主权一揽子方案”，提出加强人工智能、半导体、云计算和开源等领域能力，以期增强欧洲数字自主性和韧性。

观点重申：

GLM5.2显示智谱模型迭代顺利，领先coding能力将维持议价能力。智谱自 GLM-4.5 起即将 Coding 锁定为核心方向，连续多代版本在该方向上专门优化，形成了较厚的工程护城河，GLM5.2强化了国产模型追赶海外模型的信心。智谱 GLM 自 5 系列发布以来已多次上调 API 价格，期间调用量不降反升，表明价格弹性较低。

国内模型公司的 ARR 仍处于早期阶段。 参照 Anthropic 的 ARR 增长曲线（23年底/24年底/25年底的ARR分别为 1/10/90 亿美元，26年5月初ARR超440亿美元），国内头部模型公司目前仍处于该曲线的起步段。主权AI趋势下，海外头部模型短期内难以直接进入中国大陆市场，而国内模型则凭借性价比优势快速渗透海外市场。这意味着国内模型的市场空间，同时来自a)国内付费订阅渗透率提升+b)海外平价替代。

投资建议：我们看好智谱作为中国大模型代表性公司参与全球竞争的潜力，并认为其 Coding、Agent 与长程任务能力将随模型迭代持续提升。后续关注算力合作进展、模型进展等，维持公司“买入”评级。

风险提示：模型竞争加剧；算力供应不足；地缘政治风险。
